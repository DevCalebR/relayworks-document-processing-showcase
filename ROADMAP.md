# Roadmap

This public roadmap is intentionally conservative. It describes product directions that are reasonable for buyers to evaluate, not guaranteed future functionality.

## Current focus

- PDF-first document ingestion.
- Local and self-hosted processing.
- Markdown, HTML, JSON, plain text, report, image, and ZIP package outputs.
- FastAPI backend and Next.js frontend source-code delivery for paid buyers.

## Evaluation priorities

- Improve public documentation for setup expectations.
- Add a verified Gumroad product URL once published.
- Add a verified public demo video link once available.
- Keep screenshots current with the paid product workflow.

## Not currently claimed

- Hosted SaaS accounts.
- Production authentication.
- Billing.
- Cloud storage.
- Multi-tenant administration.
- Enterprise compliance guarantees.
- Perfect PDF extraction or guaranteed OCR accuracy.
