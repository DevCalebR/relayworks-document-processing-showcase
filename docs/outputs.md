# Outputs

RelayWorks Document Processing Kit is designed to turn PDFs into AI-ready output packages.

## Output formats

- Markdown for readable, model-friendly document content.
- HTML for rendered review and downstream display.
- Structured JSON for programmatic ingestion.
- Plain text for simple search, indexing, or model input.
- Selected goal output for workflow-specific document preparation.
- Processing report for inspectable conversion results.
- Extracted images where available from the PDF workflow.
- Downloadable ZIP packages for handoff or storage.

Actual outputs depend on the content and structure of the source PDF.

## Common downstream uses

- RAG ingestion.
- Knowledge-base preparation.
- Internal AI workflow inputs.
- Document conversion review.
- Developer handoff packages.

## Extraction expectations

PDF extraction quality depends on the source PDF and conversion tooling. Complex layouts, scans, tables, figures, and unusual encodings may require review or additional handling.
