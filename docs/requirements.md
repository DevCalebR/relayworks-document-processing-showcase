# Requirements

Buyers should be comfortable running local developer tools.

## Public requirements summary

- macOS or Linux recommended.
- Python 3.11 recommended for the FastAPI backend.
- Node.js 20 or 22 recommended for the Next.js frontend.
- npm.
- Marker installed separately and available to the local environment.
- Local filesystem access for uploads, generated outputs, and downloadable packages.

RelayWorks is intended for developers and technical teams. It is not positioned as a one-click hosted SaaS account.

This public overview intentionally does not reproduce the full commercial installation guide delivered with the paid source code.

## Operational notes

- Run it in an environment you control.
- Review generated outputs before using them in production AI workflows.
- Validate extraction quality with representative PDFs from your own use case.
- Treat scanned or complex-layout PDFs as higher-risk inputs.
