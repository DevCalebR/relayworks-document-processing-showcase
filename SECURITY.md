# Security policy

This repository does not run the commercial application or accept document uploads. Only the latest commit on `main` is supported for public showcase material.

Report accidental disclosure, unsafe sample content, archive traversal concerns, or a vulnerable link privately through the [DevCalebR GitHub profile](https://github.com/DevCalebR). Do not attach customer files, proprietary source, secrets, or personal data.

The paid source package has a separate security boundary. Buyers are responsible for securing deployment, storage, network access, third-party tools, and processed documents in their environment.
