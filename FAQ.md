# FAQ

## What is RelayWorks Document Processing Kit?

RelayWorks Document Processing Kit is a commercial source-code product for developers who need a self-hosted PDF-to-AI document processing workflow.

## What does the paid kit include?

The paid kit is described as complete FastAPI and Next.js source code for converting PDFs into AI-ready outputs such as Markdown, HTML, JSON, plain text, processing reports, extracted images, and downloadable packages.

## Is the paid source code included here?

No. This public repository is only a showcase and product-evaluation hub. The paid source code is distributed through Gumroad after purchase.

## Why is there no open-source license?

RelayWorks Document Processing Kit is a commercial product, not an open-source release. This showcase repository should not be treated as licensing the paid application source code.

## Can sensitive documents stay local?

The product is designed for local and self-hosted processing so buyers can keep documents on their own machine or infrastructure.

## Does it support every document format?

No. The current product is PDF-first.

## Does it guarantee perfect extraction?

No. Extraction quality depends on the PDF, layout complexity, embedded text, scans, images, tables, and the underlying conversion tooling.

## Does it include hosted SaaS functionality?

No. The current kit should not be evaluated as a hosted SaaS product with accounts, billing, cloud storage, or multi-tenant operations.
